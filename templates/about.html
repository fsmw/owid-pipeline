About OWID Dataset Cleaner

What is this?

OWID Dataset Cleaner is a web application that helps researchers, analysts, and data enthusiasts easily search, filter, and download datasets from Our World in Data.

Features

  • 🔍 Search through all OWID datasets by keywords
  • 🌍 Filter by countries using preset groups (G7, G20, EU, etc.) or custom selection
  • 📅 Filter by time using preset periods or custom year ranges
  • 📊 Preview filtered data before downloading
  • ⬇️ Download cleaned CSV files ready for analysis
  • ✨ Clean UI designed for modern workflows

How to Use

  1. Search for a dataset using keywords (e.g., "climate", "population", "covid")
  2. Click on a dataset card to view details and filtering options
  3. Apply filters using preset groups or custom selections
  4. Preview the filtered data to verify it meets your needs
  5. Download the cleaned CSV file

Preset Filters

Country Groups

  • G7: United States, Canada, United Kingdom, France, Germany, Italy, Japan
  • G20: All G20 member countries
  • EU: European Union member states
  • BRICS: Brazil, Russia, India, China, South Africa
  • Latin America: Latin American countries
  • Africa: African countries
  • Asia-Pacific: Asia-Pacific countries

Time Periods

  • Last 5 years
  • Last 10 years
  • Last 20 years
  • 21st Century: 2000 to present
  • By decade: 2020s, 2010s, 2000s

Data Source

All data comes from Our World in Data, a scientific online publication that focuses on large global problems such as poverty, disease, hunger, climate change, war, existential risks, and inequality.

Technology Stack

  • Backend: Flask (Python)
  • Frontend: Tailwind CSS + Alpine.js
  • Data Processing: Pandas
  • Data Source: OWID GitHub Repository

Note: This is an independent tool and is not officially affiliated with Our World in Data. We simply make their excellent data more accessible through filtering and cleaning utilities.
